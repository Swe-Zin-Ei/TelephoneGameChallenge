# Telephone Game Challenge

A browser-based telephone game with voice recognition, text-to-speech, and achievement certificates.

🔴 **Live Demo:** https://swe-zin-ei.github.io/TelephoneGameChallenge/

---

## 🎮 How to Play

1. Enter your name
2. Choose voice type and English level
3. Pick a category (Easy, Medium, Hard, Funny, Movies)
4. Click Start Game
5. Remember the message, then type or speak it
6. See how accurate you are!

---

## ⌨️ Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Ctrl+Enter` | Submit answer |
| `Space` | Start speaking |
| `N` | Next round |
| `R` | Restart game |
| `S` | Start/Pause |
| `Esc` | Close modals |
| `?` | Show help |

---

## 🏆 Certificates

- First Steps - Play first game
- Sharp Memory - 80% accuracy
- Perfect Recall - 90% accuracy
- Photographic Memory - 100% accuracy
- Enthusiast - 5 games
- Veteran - 10 games
- Legend - 25 games

---

## 🛠️ Technologies

- HTML5, CSS3, JavaScript ES6+
- Web Speech API (voice recognition + TTS)
- Web Audio API (sound effects)
- localStorage (user profiles, game history)
