The Telephone Game

A dinner party game gone digital. We transformed the classic "Chinese Whispers" into a modern voice-recognition challenge that tests memory, listening skills, and communication accuracy.

The traditional telephone game has been entertaining people for generations. Now, with the power of modern web technologies, we bring this experience to your browser with real-time voice recognition, text-to-speech, and detailed accuracy tracking.

ABOUT US

What We Built

A fully functional browser-based telephone game with real voice recognition and text-to-speech capabilities. Built using HTML5, CSS3, and vanilla JavaScript.

Players can choose their English proficiency level (B1 to C2) which affects speech speed, select from 5 unique voice types, and track their progress through an achievement certificate system.

The game features 5 word categories with over 85 unique sentences, persistent user profiles using localStorage, dark mode toggle, keyboard shortcuts, and auto-save functionality.

Created as part of the Web Technologies course at Edinburgh Napier University.
